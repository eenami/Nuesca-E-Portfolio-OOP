# 📘 Midterm Project: E-Portfolio (Object-Oriented Programming)
Visit site: https://eenami.github.io/Nuesca-E-Portfolio-OOP/index.html

## 📌 Project Overview
This project is an E-Portfolio created as part of the Midterm requirement for Object-Oriented Programming. It showcases different Java activities, exercises, and reflections that demonstrate understanding of OOP concepts.

## 🎯 Objectives
- Apply Object-Oriented Programming concepts in Java
- Demonstrate understanding of variables, methods, operators, and control structures
- Practice problem-solving and code tracing
- Organize programming outputs and reflections in a structured portfolio

## 📚 Topics Covered
- Introduction to Java
- Java Program Structure
- Variables and Data Types
- Methods (Void and Non-void)
- Operators (Unary, Arithmetic, Relational, Logical, etc.)
- Control Structures (if-else, switch, loops)
- Problem Solving and Code Tracing

## 🧠 Included Activities
- Midterm Quizzes
- Midterm Seatworks
- Midterm Assignments
- Midterm Activities
- Midterm Exam

## 💡 Key Learnings
This project helped improve understanding of how Java works as a structured and object-oriented language. It strengthened skills in logical thinking, debugging, and building real-world programs using proper programming concepts.
